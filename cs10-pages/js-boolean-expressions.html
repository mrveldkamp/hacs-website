Boolean Data Type

JavaScript has a Boolean Data Type that can be one of two possible values: true or false.

Boolean Data Type = true or false

Boolean Expressions

Just as mathematical expressions evaluate to a Number data type, boolean expressions evaluate to a Boolean data type.

Boolean Expressions are expressions that evaluate to true or false. 

    age < 18 // evaluates to true or false

Just as mathematical expressions are made up of mathematical operators, boolean expressions are made up of comparison and logical operators.

Comparison Operators

Comparison operators are used to build boolean expressions:

  • Equal To: ==
  • Not Equal To: !=
  • Greater Than: >
  • Greater Than or Equal To: >=
  • Less Than: <
  • Less Than or Equal To: <=

Examples of Comparison Operators


    // Equal To (==)
    5 == 8 evaluates to false
    5 == 5 evaluates to true

    // Not Equal (!=)
    5 != 8 evaluates to true
    8 != 8 evaluates to false

    // Greater Than (>)
    5 > 8 evaluates to false
    12 > 8 evaluates to true

    // Greater Than or Equal To (>=)
    8 >= 8 evaluates to true
    7 >= 8 evaluates to false

    // Less Than (<)
    5 < 8 evaluates to true
    9 < 8 evaluates to false

    // Less Than or Equal To (<=)
    8 <= 8 evaluates to true
    2 <= 8 evaluates to true

Logical Operators

Logical Operators allow you to build more complex boolean expressions by connecting boolean expressions:

  • Logical And: &&
  • Logical Or: ||
  • Logical Not: !

Examples of Logical Operators


    // And (&&) is true if all connected boolean expressions are true
    5 < 8 && 20 > 10 evaluates to true
    5 < 2 && 20 > 10 evaluates to false

    // Or (||) is true if at least one of the connected boolean expressions is true
    5 < 2 || 20 > 10 evaluates to true
    5 < 2 || 7 > 10 evaluates to false
    1 < 2 || 20 > 10 evaluates to true

    // Not (!) changes true to false and false to true
    !(3 < 5) evaluates to false
