Data Types

Two main types of data in JavaScript are the String data type and the Number data type.

String Data Type

A String is a sequence of characters used to represent text. Strings are enclosed by double or single quotes.


    "Hello World", 'green', '42'

Number Data Type

A Number is simply a number with or without a decimal and not enclosed by quotes.


    25, 3.14159, 42

Data Operations

JavaScript has various operations that may be used on Strings and Numbers.

String Operations

One operation on Strings is concatenation: the addition or joining of Strings.


    // Join two strings together with addition / concatenation
    'Hello' + 'World' evaluates to 'HelloWorld'

Number Operations

JavaScript supports various math operations on Numbers. Take note that JavaScript follows proper order of operations and () may be used in calculations to influence the order of operations.


    // Addition
    2 + 3 evaluates to 5

    // Subtraction
    7 - 3 evaluates to 4

    // Multiplication
    6 * 5 evaluates to 30

    // Division
    5 / 2 evaluates to 2.5

    // Remainder (Modulus)
    5 % 2 evaluates to 1

    // Exponent 
    5 ** 2 evaluates to 25

    // Order of Operations
    1 + 2 * 3 evaluates to 7
    (1 + 2) * 3 evaluates to 9

Data Conversions

JavaScript can convert one type of data to another, if possible.

Automatic Conversions

In some cases, JavaScript will convert data automatically in order to make the operation work.


    // JavaScript automatically converts the Number to a String
    2 + '3' evaluates to '23'

    // JavaScript automatically converts the String to a Number
    2 * '3' evaluates to 6

Conversion Functions

JavaScript provides the functions Number() and String() to convert data, if possible.


    // Convert Number to String
    String(23.4) evaluates to '23.4'
    String(-15) evaluates to '-15'

    // Convert String to Number
    Number('7') evaluates to 7
    2 + Number('3') evaluates to 5

    // Not possible
    Number('Hello World!') evaluates to NaN (Not a Number)
    Number("seven") evaluates to NaN (Not a Number)
