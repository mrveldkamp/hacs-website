Code on Demand

In JavaScript, code may be executed on demand by using Functions and Event Listeners.

Functions

A function may be thought of as a block of code. The code stored inside of a function will not execute until the function is invoked / called (ie. told to do its job!).

main.js 

    // Define a function that stores code
    function greeting() {
      alert('Hello Functions');
      alert('I am inside of a function.');
    }

The name of the function can be anything, but should represent the task of the function.

Function names generally start with a lowercase letter and may include lower and upper case letters, numbers and underscores.

Event Listeners

Event listeners may be added to html elements. Event listeners will tell a specified function to do its job when a certain event occurs.

main.js 

    // Run greeting function when document is clicked
    document.addEventListener('click', greeting);

Types of events to listen for include:

  • 'click'
  • 'dblclick'
  • 'mousedown'
  • 'mouseup'
  • 'scroll'
  • 'keydown'
  • 'keyup'
  • ...
