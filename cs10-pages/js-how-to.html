JavaScript

JavaScript is a programming language that is most well known as a scripting language for web pages. JavaScript helps make web pages alive! Programs in JavaScript may be run by a web browser and are commonly called scripts.

Internal Script


<!DOCTYPE html>
<body>
  <h1>JavaScript Concept Notes</h1>

  <!-- Run JavaScript Commands -->
  <script>
    alert('Hello World!');
    alert('This is awesome!');
  </script>
</body>

Script is embedded into an HTML document.

JavaScript commands execute inside of the <script></script> element.

alert() is a JavaScript command that tells the browser to display a provided message in a pop-up window.

External Script

index.html 

<!DOCTYPE html>
<body>
  <h1>JavaScript Concept Notes</h1>

  <!-- Run JavaScript Commands -->
  <script src='main.js'></script>
</body>
main.js 

      // JavaScript File 

      alert('Hello World!');
      alert('This is awesome!');

Script is loaded into HTML document from an external file.

Note the use of // for JavaScript comments.

Note: Browser console (F12) is useful for checking error messages for your JavaScript code. You can also log messages to the console using the JavaScript command: console.log()
