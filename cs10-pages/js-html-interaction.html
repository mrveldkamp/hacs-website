JavaScript & HTML

JavaScript may be used to interact with a web page. JavaScript can select html elements and then modify their content, style and behaviour.

Select html elements...


  document.getElementById('id');

Add event listeners to a specific html element...


  document.getElementById('id').addEventListener('event', functionName);

Set content of a specific html element...


  document.getElementById('id').innerHTML = '______';

Set attribute of a specific html element...


  document.getElementById('id').attr = '______';
  // Replace .attr with any valid html attribute: (src, href, ...)

Set style property of a specific html element...


  document.getElementById('id').style.property = '______';
  // Replace .property with any valid CSS property: (color, width, ...)
  // CSS properties with hyphens change to camelCasing (ie. font-size -> fontSize)

Add / remove css classes...


  document.getElementById('id').classList.add('className');
  document.getElementById('id').classList.remove('className');
  document.getElementById('id').classList.toggle('className');
