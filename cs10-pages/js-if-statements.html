Selection Control Structure

If statements are a type of selection control structure. They allow you to control / select what code to execute.

Binary Selection

Select one of two paths...


    if (boolean_expression) {
      console.log('boolean_expression is true');
    } else {
      console.log('boolean_expression is false');
    }

Unary Selection

Select path or not...


    if (boolean_expression) {
      console.log('boolean_expression is true');
    }

Chained Selection

Select one of multiple paths...


    if (boolean_expression_1) {
      console.log('boolean_expression_1 is true');
    } else if (boolean_expression_2) {
      console.log('boolean_expression_2 is true');
    } else if (boolean_expression_3) {
      console.log('boolean_expression_3 is true');
    } else {
      console.log('All boolean_expressions are false');
    }

    // Note: insert as many else if conditions as needed.

Chained Selection without Else

Select one or none of multiple paths...


    if (boolean_expression_1) {
      console.log('boolean_expression_1 is true');
    } else if (boolean_expression_2) {
      console.log('boolean_expression_2 is true');
    } else if (boolean_expression_3) {
      console.log('boolean_expression_3 is true');
    } 

    // Note: insert as many else if conditions as needed.

Nested If Statements

If statements may be nested inside of other if statements.


    // Outer Binary If Statement
    if (boolean_expression_1) {
      // Inner Binary If Statement
      if (boolean_expression_2) {
        console.log('1 true, 2 true');
      } else {
        console.log('1 true, 2 false');
      }
    } else {
      // Inner Chained If Statement
      if (boolean_expression_3) {
        console.log('1 false, 3 true');
      } else if (boolean_expression_4) {
        console.log('1 false, 3 false, 4 true');
      } else {
        console.log('1 false, 3 false, 4 false');
      }
    }
