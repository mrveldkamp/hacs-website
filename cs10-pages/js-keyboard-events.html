What Key was That?

Event listeners will send information about an event to the specified event function. This data can be stored and accessed by an event variable that is declared inside of the event function header.

For key events, the data that is sent from the event listener includes an event.keyCode property that stores the key code of the key that was pressed (eg. A key code of 32 is the 'Space Bar').

To test the key code for a specific key, simply log the event.keyCode to the console:


    // Event Listener
    document.addEventListener('keydown', keydownHandler);

    // Event Function
    function keydownHandler(event) {
      console.log(event.keyCode);
    }
