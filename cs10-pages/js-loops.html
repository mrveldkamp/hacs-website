Introduction

Loops are used to repeat code. Two common loops in JavaScript are the while loop and the for loop.

While Loop

The code inside a while loop will repeatedly execute as long as the condition is true.


    while (condition) {
      // Code block 
      // to be 
      // executed.
    }

Example: Log the numbers 1 to 100 to the console...


    let num = 1;
    while (num <= 100) {
      console.log(num);
      num++;
    }

For Loop

The code inside a for loop will repeatedly execute as long as the condition is true. The flow of control of a for loop is as follows:

  1. execute init statement (often used to initialize variables)...
  2. test condition...
  3. if test condition is true, execute loop body...
  4. execute update statement (often used to update variables)...
  5. repeat 2 - 4 until test condition is false...

    for (init; condition; update) {
      // Code block 
      // to be 
      // executed.
    }

Example: Log the numbers 1 to 100 to the console...


    for (let num = 1; num <= 100; num++) {
      console.log(num);
    }

Repeat Code

Loops may be used to repeat code:

  • a set number of times,
  • or an unknown number of times.

For Loops often work best when repeating code a set number of times:

Example: Simulate flipping a coin 500 times


    // Coin Flip Simulator
    let numHeads = 0;
    let numTails = 0
    let randNum;

    // Loop 500 times
    for (let n = 0; n < 500; n++) {
      // Simulate Coin Flip 50/50 chance
      randNum = Math.random();
      if (randNum < 0.5) {
        numHeads++;
      } else {
        numTails++;
      }
    }

    // Output Final Results
    console.log("Heads: " + numHeads + " Tails: " + numTails);

While loops often work best when repeating code an unknown number of times.

Example: Simulate flipping a coin until 100 heads.


      // Coin Flip Simulator
      let numHeads = 0;
      let numTails = 0
      let randNum;
  
      // Loop until 100 heads
      while (numHeads < 100) {
        // Simulate Coin Flip 50/50 chance
        randNum = Math.random();
        if (randNum < 0.5) {
          numHeads++;
        } else {
          numTails++;
        }
      }
  
      // Output Final Results
      console.log("Heads: " + numHeads + " Tails: " + numTails);

Nested Loops

Loops may be nested inside of other loops...


      // Outer Loop
      for (init1; condition1; update1) {
        // Inner Loop
        for (init2; condition2; update2) {
          ...
          ...
        }
      }
