Numerical Input

Input received from prompt() or <input> elements is always returned as a String data type.

In order to do mathematical operations with input it is best to convert the input to a Number data type by using the Number() function.


    // Prompt user for numerical input
    let num1 = Number(prompt("Please enter a number."));
    
    // Get numerical data from input element
    let num2 = Number(document.getElementById('numInput').value);

IPO Algorithm

A common algorithm is an Input, Process, Output algorithm. This algorithm involves:

  1. Input - get input from the user.
  2. Process - process input in some way (ie. calculations).
  3. Output - output the result.

Example: Simple Addition Calculator


    // SIMPLE ADDITION CALCULATOR

    // Input - two numbers
    let num1 = Number(prompt('Please enter a number:''));
    let num2 = Number(prompt('Please enter another number:'));

    // Process - calculate sum
    let total = num1 + num2;

    // Output - display result
    alert('The sum of your number is ' + total + '.');
