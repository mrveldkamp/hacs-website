Random Numbers

The JavaScript Math object may be used to generate random numbers:

  • Math.random() returns a random decimal from 0 up to 1.
  • Math.floor() rounds a value down to the nearest integer.

    // Random Decimal between 0 (inclusive) and 1 (exclusive): 0 to 0.9999...
    Math.random();
    
    // Random Decimal between 0 (inclusive) and 10 (exclusive): 0 to 9.9999...
    Math.random() * 10;

    // Random Integer between 0 (inclusive) and 10 (exclusive): 0, 1, ..., 8, 9
    Math.floor(Math.random() * 10);

    // Random Integer between 1 (inclusive) and 11 (exclusive): 1, 2, ..., 9, 10
    Math.floor(Math.random() * 10) + 1

Simulators

Random Numbers allow us to use randomness and probability to simulate the result of an event:


    // Simulate result of flipping a coin - 50/50 chance
    let randNum = Math.random();
    if (randNum < 0.5) {
      alert("Heads!");
    } else {
      alert("Tails!");
    }
