Operations to Update Variables

The following operations are useful when you wish to add or subtract values from a variable. Take note of the optional shorthand notations.


    // Increase Variable by 5
    x = x + 5  or  x += 5

    // Decrease Variable by 5
    x = x - 5  or  x -= 5

    // Increase Variable by 1
    x = x + 1  or  x += 1  or  x++
    
    // Decrease Variable by 1
    x = x - 1  or  x -= 1  or  x--

    // Append "!" to string variable
    name = name + '!'  or  name += '!'

Short hand notations also exist for multiplication, division and modulus.


    x = x * 5  or  x *= 5
    x = x / 5  or  x /= 5
    x = x % 2  or  x %= 2
