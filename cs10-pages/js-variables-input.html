Variables

Variables are used to store data.


    // Declare a variable - tell JS to make a storage container called age 
    let age;

    // Assign data to variable - put data into age container 
    age = 15;

    // Use variable - log the contents of the age variable to the console
    console.log(age); // Will log 15 to the console

    // Reassign variable - put new data into container
    age = 16;
    console.log(age); // Will log 16 to the console

It is possible to declare a variable and assign a value to it in one line.


    let age = 15;

Take note of some rules and conventions to follow when naming a variable:

  • Begin variable names with a lower-case letter.
  • Only use letters (a-z, A-Z), numbers(0-9) and underscores(_).
  • Use came casing for multi word variables (eg. myCamelCaseVariable)
  • Make variable names intuitive: describe what they store.
  • Avoid using JavaScript reserved words (eg. let, function, alert, ...)

Text Input

Text input from the user may be obtained with the prompt() function or with <input> elements. The text input from the user may be stored in variables for later use.

prompt()

The prompt() function will open a pop-up window where the user can provide input. 


    // Ask user for their favourite sport and give feedback
    let faveSport = prompt("Please enter your favourite sport.");
    alert("The sport of " + faveSport + " is fascinating!");

<input> elements

Add an <input> element with an id into your html document.


    <-- Favourite Sport Input -->
    <p>Favourite Sport: <input id='faveSportInput'></p>

Use JavaScript to select the input element by id and get its value property.


    let faveSport = document.getElementById('faveSportInput').value;
    alert("The sport of " + faveSport + " is fascinating!");
