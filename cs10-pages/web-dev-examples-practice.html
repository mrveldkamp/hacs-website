Create from Given Design

The main goal is to use HTML and CSS skills to create a simple web page from a given design.

The web page design will come in a folder that includes all necessary content (images, sounds, text, etc.) and a README file with:

  • a screenshot of what the final web page should look like,
  • and some style notes regarding fonts, colors, etc.

The Video Lessons will demo the process of using HTML and CSS to create a web page from a given design and then there are some Assignments to test your skills.
