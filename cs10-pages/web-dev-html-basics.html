HTML - Web Page Content

The basic concepts of HTML that you should become familiar with are:

  • Page Structure & Tags: HTML Intro
  • Headings: <h1>...<h6>
  • Paragraphs: <p>
  • Horizontal Rules: <hr>
  • Images: <img>
  • Links: <a>
  • Lists: <ul>, <ol>
  • Audio: <audio>
  • Buttons: <button>
  • Input Elements: <input>

The links behind each concept take you to reference notes from w3schools.

The Video Lessons tab provides video lessons on each concept.
