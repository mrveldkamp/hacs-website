Overview

Big Idea

The big idea of "Intro to Web Development" is using HTML and CSS to create a web page from a given design.

HTML stands for Hyper Text Markup Language and is used to describe the structure and content of a web page using various markup tags.

CSS stands for Cascading Style Sheet. CSS is used to apply styles such as: colors, fonts, borders, margins, etc. to html elements.

What is a Web Page?

A web page consists of a bunch of files that get displayed by a web browser.

A simple web page will usually include the following files:

images folder...

...contains the image files to be used in the web page.

styles.css...

...contains the style rules that control things like color, fonts, layout, etc.

index.html...

...contains the content of a web page and is the document that the web browser will open. This document tells the browser how to put together the content, styles and images for the web page.

Publishing a Web Page

Website files are hosted on web servers. Upon request, the servers "serve" the files to individual computer users. The Internet makes it all possible.

It is recommended to develop and test web pages on a local computer before uploading them to a public server. This local development will be the main focus of this course.
