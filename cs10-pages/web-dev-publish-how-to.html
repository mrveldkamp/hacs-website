Github Pages

Github Pages is a free service that may be used to host your projects on the web.

  • Create a GitHub account or login if you already have one.
  • Create a public repository and upload your files into the repository.
  • Go to the repositories Settings page and turn on Github Pages!
  • View your web page at the address/url provided by Github Pages.

The Video Lessons tab has a video that walks you through this process.
