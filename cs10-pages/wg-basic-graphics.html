HTML Canvas Element

HTML has a <canvas> element that may be added to your web page. The canvas element may be styled with css like any other element.

Canvas Coordinate System

The canvas uses a coordinate system to specify the location of items to draw. The origin is the top-left corner of the canvas and the x-axis increases as you go right from the origin and the y-axis increases as you go down from the origin.

Graphics Context

A graphics context is used to draw on the canvas. The graphics context for the canvas contains many methods for drawing shapes, text and images onto the canvas.
