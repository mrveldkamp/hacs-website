Built-in Methods

JavaScript has some built-in tools that help when working with arrays. A few of these tools are highlighted below...

includes() Method

The includes() method determines whether an array contains a specified item. This method returns true if the array contains the item, and false if not.

array.includes(item)


    let attendanceList = ["Angela", "Bryan", "Carol", "David"];
    if (attendanceList.includes("Joe")) {
      console.log("I'm so glad Joe could make it!");
    } else {
      console.log("I wonder where Joe is?");
    }

indexOf() Method

The indexOf() method searches the array for the specified item, and returns its position. Returns -1 if the item is not found. If the item is present more than once, the indexOf() method returns the position of the first occurence. (Note: there is also a lastIndexOf() method that returns the position of the last occurence.)

array.indexOf(item)


    let attendanceList = ["Angela", "Bryan", "Carol", "David"];
    let index = attendanceList.indexOf("Carol");
    if (index == -1) {
      console.log("Carol not found.");
    } else {
      console.log("Carol found at position: " + index);
    }

Math.max() Method

The Math.max() method takes in numerical arguments and returns the argument with the highest value. (Note: there is also a Math.min() method.)

Math.max(n1, n2, n3, ..., nX)

The Math.max() method is designed to work with multiple arguments and not an array. In order to use Math.max() for arrays, the spread operator (...) may be used to expand the array in place of the required arguments.


    let maxNum = Math.max(50, 20, 30, 70, 40); // maxNum -> 70

    let numArray = [2, 8, 12, 5, 9, 20, 17, 3];
    let maxElement = Math.max(...numArray); // maxElement -> 20
    let minElement = Math.min(...numArray); // minElement -> 2
