Data Structures - Associative Arrays

An associative array is a data structure that has named indexes instead of the numerical indexes a regular array uses.

An associative array may be described as a collection of key-value pairs.

In JavaScript, objects behave like associative arrays.

Values in the object may be accessed using dot notation or [ ] notation.


    // Object Literal (Associative Array)
    let student = {name: "Megan", id: 1, grade: 10};

    student.grade = 11; // Access grade property with dot notation
    student["grade"] = 11; // Access grade property with [ ] notation

It is also possible to add key-value pairs with dot notation or [ ] notation.


    student.username = "megan105"; // Add username property with dot notation
    student["username"] = "megan105"; // Add username property with [ ] notation
