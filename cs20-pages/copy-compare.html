Copy Arrays

The slice() method may be used to copy all or part of an array. The slice() method takes in optional start (inclusive) and stop (exclusive) arguments.


    // ARRAY SLICE METHOD
    let anArray = [10, 20, 30, 40, 50, 60, 70];
    let slice1 = anArray.slice(1, 4); // index 1 to 3: [20, 30, 40]
    let slice2 = anArray.slice(3); // index 3 to end: [40, 50, 60, 70] 
    let slice3 = anArray.slice(); // entire array: [10, 20, 30, 40, 50, 60, 70]

Compare Arrays

To compare simple arrays, one technique is to convert the arrays to strings and then check the strings for equality. The join() method may be used to convert an array to a string, using a provided argument as the separator between elements.


    // ARRAY COMPARE WITH JOIN
    let array1 = [1, 2, 3];
    let array2 = [1, 2, 3];
    let array3 = [3, 2, 1];
    array1 == array2; // False (Different memory addresses)
    array1.join("") == array2.join(""); // True ("123" == "123")
    array1.join("") == array3.join(""); // False ("123" != "321")
