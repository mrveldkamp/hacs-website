Data Options

Some possible ways of using data structures for data storage include:

  • Parallel Arrays
  • 2D Arrays
  • Arrays of Objects

Parallel Arrays

The example below uses parallel arrays to store the centers of three bubbles. The data for the first bubble is stored at index 0 of each array, the second bubble is stored at index 1 of each array, and the third bubble is stored at index 2 of each array.


    // BUBBLE CENTERS WITH PARALLEL ARRAYS
    let xVals = [200, 400, 600];
    let yVals = [300, 300, 300];

    // First bubble center at (xVals[0], yVals[0])
    // Second bubble center at (xVals[1], yVals[1])
    // Third bubble center at (xVals[2], yVals[2])

2D Arrays

The example below uses a 2D array to store the centers of three bubbles. The data for the first bubble is stored in the array located at index 0, the second bubble is stored in the array at index 1, and the third bubble is stored in the array at index 2.


    // BUBBLE CENTERS WITH 2D ARRAY
    let bubbles = [ [200, 300], [400, 300], [600, 300] ];
    
    // First bubble center at (bubbles[0][0], bubbles[0][1])
    // Second bubble center at (bubbles[1][0], bubbles[1][1])
    // Third bubble center at (bubbles[2][0], bubbles[2][1])

Arrays of Objects

The example below uses an array of objects to store the centers of three bubbles. The data for the first bubble is stored in the object located at index 0, the second bubble is stored in the object at index 1, and the third bubble is stored in the object at index 2.


    // BUBBLE CENTERS WITH ARRAY OF OBJECTS
    let bubbles = [ {x: 200, y: 300}, {x: 400, y: 300}, {x: 600, y: 300} ];
    
    // First bubble center at (bubbles[0].x, bubbles[0].y)
    // Second bubble center at (bubbles[1].x, bubbles[1].y)
    // Third bubble center at (bubbles[2].x, bubbles[2].y)

Make it Snow

Instructions

Create a program to simulate falling snow.

  • Each snowflake should have its own location, size and speed.
  • When a snowflake goes off the bottom of the canvas it should teleport to a random location above the canvas.
  • Implement the ability to add and remove snowflakes on mouse or key events.
