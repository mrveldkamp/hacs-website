Define & Invoke

A function is defined as a named block of code. The code inside of a function will not execute until the function is invoked by name.


      // Define a Function
      function displayGreeting() {
        console.log('Hello.');
        console.log('Have a good day!');
      }

      // Invoke Function 3 times
      displayGreeting();
      displayGreeting();
      displayGreeting();

Caution! ... Common Mistakes

  • Remember to have starting and closing braces, { }, when defining a function.
  • Don't forget the ( ) after the function name when invoking the function.

Pass Data into a Function

A function may be given data to work with. Arguments refer to the values passed into a function and parameters refer to the variables that store the argument values.


      // Define function with parameters to store a name and time of day
      function displayGreeting(name, timeOfDay) {
        console.log('Hello ' + name + '.');
        console.log('Have a good ' + timeOfDay + '!');
      }

      // Invoke function with different argument values
      displayGreeting('Megan', 'Morning'); // -> Hello Megan. Have a good Morning!
      displayGreeting('Azra', 'Afternoon'); // -> Hello Azra. Have a good Afternoon!
      displayGreeting('Nathan', 'Night'); // -> Hello Nathan. Have a good Night!
