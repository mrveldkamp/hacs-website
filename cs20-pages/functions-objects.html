Object Basics

It is common to have many variables that are all related to each other. For example, a circle may have variables for its center (x,y), radius, colour, etc.

In JavaScript, it is possible to store related information in a data structure called an object: a collection of named values. These named values are called properties of the object.

Objects allow the creation of a single variable that can store multiple properties.

Create an Object

You can define a JavaScript object with an object literal. Place comma separated name-value pairs inside of { }. 


      // Object (collection of named values)
      let circle = {x: 300, y: 200, radius: 25, color: 'grey'};

To improve readability, it is common to have each property on its own line.


      // Object (collection of named values)
      let circle = {
        x: 300,
        y: 200,
        radius: 25,
        color: 'grey'
      };

Access Object Properties

A common way to access object properties is with dot notation.


      // DRAW CANVAS
      ctx.fillStyle = circle.color;
      ctx.beginPath();
      ctx.arc(circle.x, circle.y, circle.radius, 0, 2 * Math.PI);
      ctx.fill();

Functions and Objects

When it comes to functions, one use of objects is to use objects as arguments and return values.

Without Objects


      // WITHOUT OBJECTS
      let circleX = 300;
      let circleY = 200;
      let circleRadius = 25;
      let circleColor = 'grey';

      // Invoke Function
      drawCircle(circleX, circleY, circleRadius, circleColor);

      // Define Function
      function drawCircle(x, y, radius, color) {
        ctx.fillStyle = color;
        ctx.beginPath();
        ctx.arc(x, y, radius, 0, 2 * Math.PI);
        ctx.fill();
      }

With Objects


      // WITH OBJECTS
      let circle = {
        x: 300,
        y: 200,
        radius: 25,
        color: 'grey'
      };

      // Invoke Function
      drawCircle(circle);

      // Define Function
      function drawCircle(aCircle) {
        ctx.fillStyle = aCircle.color;
        ctx.beginPath();
        ctx.arc(aCircle.x, aCircle.y, aCircle.radius, 0, 2 * Math.PI);
        ctx.fill();
      }
