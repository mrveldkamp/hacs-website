2d Arrays

A 2D Array is an "array of arrays".

An outer array will serve as a container for other arrays.


      let array2D = [ [255, 255, 0], [0, 255, 0], [0, 255, 255] ];

A double index operator, [][], may be used to access values inside the 2D array.


      // Access the 2nd value in the 3rd inner array
      array2D[2][1] = 0; // -> [ [255, 255, 0], [0, 255, 0], [0, 0, 255] ]

Grids

A common use of 2D arrays is to represent a grid with rows and columns.


      let grid = [ [0, 1, 0],
                  [1, 1, 0],
                  [0, 0, 1] ];

Each inner array is a row and the columns are the individual elements inside of the inner arrays.

To access elements in the grid, use grid[row][column]


      // Example: Access 3rd element in the first row
      grid[0][2] = 1;
