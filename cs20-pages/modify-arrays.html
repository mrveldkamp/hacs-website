Modify Elements

Array elements may be reassigned and modified just like regular variables.


    let nums = [1, 2, 3, 4, 5, 6];
    nums[0] = 3; // Assign the 1st element to 3
    nums[1]++; // Increment the 2nd element by 1
    nums[2] *= 1; // Multiply the 3rd element by 1
    nums[3] -= 2; // Decrement the 4th element by 2
    nums[4] /= 2; // Divide the 5th element by 2

Add & Remove Elements

There are some useful JavaScript array methods that help to add and remove array elements.

push()


    // Add item(s) to the end of the array
    anArray.push(item1, item2, ...);

pop()


    // Remove last element
    anArray.pop();
    
    // Remove and store last element
    let lastElement = anArray.pop();

unshift()


    // Add item(s) to the beginning of the array
    anArray.unshift(item1, item2, ...);

shift()


    // Remove first item
    anArray.shift();
    
    // Remove and store first item
    let firstElement = anArray.shift();

splice()


    // Add and/or Remove array elements at a specific index
    anArray.splice(index, howManyToRemove, item1, item2, ...);

    // Remove and store third item
    let thirdItem = anArray.splice(2, 1);

    // Add item to index 5
    anArray.splice(5, 0, item);
