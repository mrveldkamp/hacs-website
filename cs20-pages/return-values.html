Return Values

A function may return a value. Simply use the return keyword followed by the value to return. The return command stops execution of the function and returns the value to where the function was called.


      // Define a random decimal function
      function randomDec(low, high) {
        // Return a random decimal between low (inclusive) and high (exclusive)
        return Math.random() * (high - low) + low;
      }

      // Invoke function and use return value in various ways
      let xVal = randomDec(0, 300); // Store returned value in a variable
      console.log(randomDec(1, 10)); // Pass returned value into console.log
      let circleArea = Math.PI * randomDec(1, 50) ** 2; // Use returned value in an expression
